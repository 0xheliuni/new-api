# 首尾帧生成视频

## OpenAPI Specification

```yaml
openapi: 3.0.1
info:
  title: ''
  description: ''
  version: 1.0.0
paths:
  /v1/videos:
    post:
      summary: 首尾帧生成视频
      deprecated: false
      description: >-
        图生视频-首帧、图生视频-首尾帧、多模态参考生视频（包括参考图、视频、音频）为 3 种互斥场景，不可混用

        多模态参考生视频可通过提示词指定参考图片作为首帧/尾帧，间接实现“首尾帧+多模态参考”效果。若需严格保障首尾帧和指定图片一致，优先使用图生视频-首尾帧（配置
        role 为 first_frame/last_frame）


        图生视频-首帧 

        字段role取值：需要传入1个 image_url 对象，字段 role 为 first_frame 或不填。

        图生视频-首尾帧

        字段role取值：需要传入2个image_url对象，且字段 role 必填。
            首帧图片对应的字段 role 为：first_frame
            尾帧图片对应的字段 role 为：last_frame
        图生视频-参考图 

        字段role取值：必填，每张参考图对应的字段 role 均为：reference_image
      tags:
        - 视频模型/SeeDance2.0
      parameters:
        - name: Authorization
          in: header
          description: ''
          required: false
          example: sk-6TgQZm
          schema:
            type: string
      requestBody:
        content:
          application/json:
            schema:
              type: object
              properties:
                model:
                  type: string
                  title: 您需要调用的模型
                prompt:
                  type: string
                  title: 输入给模型的提示词信息
                content:
                  type: array
                  items:
                    type: object
                    properties:
                      type:
                        type: string
                        description: 此处应为 image_url
                        title: 输入内容的类型
                      role:
                        type: string
                        title: 输入给模型的图片对象。
                      image_url:
                        type: object
                        properties:
                          url:
                            type: string
                            description: 图片 URL：填入图片的公网 URL
                            title: 图片 URL 、图片 Base64 编码
                        required:
                          - url
                        x-apifox-orders:
                          - url
                        title: 输入给模型的图片对象。
                    required:
                      - type
                      - role
                      - image_url
                    x-apifox-orders:
                      - type
                      - role
                      - image_url
                  description: 支持文本、图片、音频、视频
                  title: 输入给模型，生成视频的信息
                return_last_frame:
                  type: boolean
                  title: 返回生成视频的尾帧图像
                  description: |-
                    默认为:false
                    true：返回生成视频的尾帧图像
                    false：不返回生成视频的尾帧图像。
                resolution:
                  type: string
                  title: 视频分辨率，枚举值
                  description: |-
                    480p, 720p, 1080p
                    Seedance 2.0 fast 不支持1080p
                ratio:
                  type: string
                  title: 输出宽高比
                  description: |-
                    支持的比例:
                    21:9, 16:9, 4:3,1:1, 3:4, 9:16
                    默认:adaptive
                    当配置 ratio 为 adaptive 时，模型会根据生成场景自动适配宽高比
                duration:
                  type: integer
                  title: 输出时长
                  description: |-
                    4~15 秒
                    默认值 5
                watermark:
                  type: boolean
                  title: 生成视频是否包含水印
                generate_audio:
                  type: boolean
                  title: 生成有声视频
                  description: 默认值 true
              required:
                - model
                - prompt
                - content
                - return_last_frame
                - resolution
                - ratio
                - duration
              x-apifox-orders:
                - model
                - prompt
                - content
                - return_last_frame
                - resolution
                - ratio
                - duration
                - generate_audio
                - watermark
            example:
              model: doubao-seedance-2-0-260128
              prompt: 镜头平稳推进，人物表情自然变化
              content:
                - type: image_url
                  role: first_frame
                  image_url:
                    url: https://example.com/start.png
                - type: image_url
                  role: last_frame
                  image_url:
                    url: https://example.com/end.png
              return_last_frame: true
              resolution: 720p
              ratio: '16:9'
              duration: 5
              watermark: false
      responses:
        '200':
          description: ''
          content:
            application/json:
              schema:
                type: object
                properties:
                  id:
                    type: string
                  task_id:
                    type: string
                  object:
                    type: string
                  model:
                    type: string
                  status:
                    type: string
                  progress:
                    type: integer
                  created_at:
                    type: integer
                required:
                  - id
                  - task_id
                  - object
                  - model
                  - status
                  - progress
                  - created_at
                x-apifox-orders:
                  - id
                  - task_id
                  - object
                  - model
                  - status
                  - progress
                  - created_at
              example:
                id: task_qghsnDbtZo7FeliS4PAA1jGQsq1X1laH
                task_id: task_qghsnDbtZo7FeliS4PAA1jGQsq1X1laH
                object: video
                model: doubao-seedance-2-0-fast-260128
                status: queued
                progress: 0
                created_at: 1778479354
          headers: {}
          x-apifox-name: 成功
      security: []
      x-apifox-folder: 视频模型/SeeDance2.0
      x-apifox-status: released
      x-run-in-apifox: https://app.apifox.com/web/project/7911422/apis/api-455612362-run
components:
  schemas: {}
  securitySchemes: {}
servers: []
security: []

```
