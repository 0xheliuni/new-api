# 私域真人人像素材资产使用指南

> 来源: <https://www.volcengine.com/docs/82379/2333589?lang=zh>

> **警告**：仅限邀测用户。需遵守《火山引擎真人认证和同人认证技术服务使用规范》。

## 与虚拟人像库的区别

真人人像库需要通过 **H5 真人认证**，从根源锁定肖像权归属。终端客户仅需完成一次授权，后续新增妆造可向同一素材组上传同人资产。

## 使用流程

### Step 1: 创建真人人像素材组（Asset Group）

1. 调用 `CreateVisualValidateSession` 生成 H5 认证页链接
2. 终端用户完成真人认证后跳转 CallbackURL
3. 解析 CallbackURL 参数获取认证结果（resultCode=10000 为成功）
4. 调用 `GetVisualValidateResult` 获取 Asset Group ID

### Step 2: 上传/管理素材（Assets）

- 入库时系统自动进行面部特征一致性比对
- 比对通过后方可入库，多人脸素材无法入库

### Step 3: 使用人像素材生成视频

使用 `asset://<asset_ID>` 格式发起视频生成任务

## API 接口

### 真人认证接口

- **CreateVisualValidateSession**: 拉起H5活体认证页（链接有效期120秒）
- **GetVisualValidateResult**: 获取认证结果及 Asset Group ID（BytedToken有效120秒）

### Asset管理接口（同虚拟人像库）

CreateAsset / ListAssets / ListAssetGroups / GetAsset / GetAssetGroup / UpdateAsset / UpdateAssetGroup / DeleteAsset / DeleteAssetGroup

### 限流

| 接口 | 限流 |
|------|------|
| CreateVisualValidateSession | 3 QPS |
| GetVisualValidateResult | 3 QPS |
| CreateAsset | 300 QPM |
| GetAsset | 100 QPS |
| 其他管理接口 | 5~10 QPS |

### CallbackURL 回调参数

- bytedToken: 认证凭证
- resultCode: 10000为成功
- algorithmBaseRespCode: 子错误码
- reqMeasureInfoValue: 是否计费(0/1)
- verify_type: 固定值real_time

### GroupType区别

- 虚拟人像：`AIGC`
- 真人人像：`LivenessFace`

### 完整使用流程

#### Step 1: 创建真人认证会话

**CreateVisualValidateSession 请求参数：**
- **CallbackURL**: 必填，认证完成后跳转URL
- **ProjectName**: 项目名称，默认`default`

**返回：**
```json
{
  "BytedToken": "202603311449168C23BA26**************",
  "H5Link": "https://h5-v2.kych5.com?...",
  "CallbackURL": "https://www.example.com/callback"
}
```

> H5Link 有效期120秒。可通过链接后缀 `lng` 参数指定语言：`zh`(简中)、`en`(英文)、`zh-Hant`(繁中)。

#### Step 2: 解析回调结果

CallbackURL 会拼接以下参数：
```
<CallbackURL>?bytedToken=xxx&resultCode=10000&algorithmBaseRespCode=0&reqMeasureInfoValue=1&verify_type=real_time
```

- **resultCode=10000**: 认证成功
- **bytedToken**: 用于GetVisualValidateResult获取Group ID

#### Step 3: 获取 Asset Group ID

**GetVisualValidateResult 请求参数：**
- **BytedToken**: 认证凭证（有效期120秒）
- **ProjectName**: 项目名称

**返回：**
```json
{"GroupId": "group-20260331145705-*****"}
```

#### Step 4: 上传素材 (CreateAsset)

与虚拟人像库相同的 CreateAsset 接口，但有额外限制：
- 系统会将上传图像与真人认证基准图像进行面部特征一致性比对
- 非同一人物的素材无法入库
- 包含多个人脸的素材无法入库

#### Step 5: 使用素材生成视频

使用 `asset://<asset_ID>` 格式，与虚拟人像库用法一致。

### 真人人像 vs 虚拟人像对比

| 维度 | 虚拟人像(AIGC) | 真人人像(LivenessFace) |
|------|---------------|---------------------|
| GroupType | AIGC | LivenessFace |
| 创建Group方式 | CreateAssetGroup | CreateVisualValidateSession+真人认证 |
| 人脸验证 | 无 | 自动面部一致性比对 |
| 肖像权 | 需自行确保 | H5真人认证锁定 |
| 多人脸素材 | 支持 | 不支持 |
| 鉴权 | AK/SK | AK/SK + ArkFullAccess权限 |

### 常见问题
**Q: 素材上传成功后无法生成视频？**
A: 素材库按项目(Project)隔离，确保 CreateAsset 和视频生成API的 ProjectName 一致。
