# Seedance 2.0 提示词指南

> 来源: <https://www.volcengine.com/docs/82379/2222480?lang=zh>

Seedance 2.0 系列模型原生支持音频与视频联合生成，拥有卓越的语义理解与多模态交互能力。

## 01 总体要领

### 1.1 文本指令的基础公式

- **主体+动作**：谁正在进行什么动作，逻辑基石
- **环境+风格**：空间背景、光影细节或特定视觉风格
- **进阶指令**：镜头调度或氛围声效等

### 1.2 多模态参考的指代控制

支持图像、音频和视频的深度参考。

- 在提示词中清晰指定参考对象，如"画面参考图片1的构图"或"动作参考视频2"
- 模型自动提取参考对象中的核心特征

## 02 文字生成

支持 T2V、I2V、R2V、V2V 场景下生成文字。

### 2.1 广告语(Slogan)

模板：

```
「文字内容」+「出现时机」+「出现位置」+「出现方式」，「文字特征」
```

### 2.2 字幕

模板：

```
画面底部出现字幕，字幕内容为"..."，字幕需与音频节奏完全同步。
```

### 2.3 气泡台词

模板：

```
「角色」说："..."，角色说话时周围出现气泡，气泡里写着台词。
```

## 03 图片参考

### 3.1 主体多视角图参考

模板：

```
参考/提取/结合+「图片n」中的「主体」，生成「画面描述」，保持「主体」特征一致。
```

- 商品类：3C数码、家居物品等
- 角色类：人物形象参考

### 3.2 多图参考

模板：

```
参考/提取/结合/按照/生成+「图片n」中的「被参考元素描述」，生成「画面描述」，保持「被参考元素」特征一致。
```

- Logo参考
- 多主体参考
- 多元素参考
- 多宫格分镜图参考
- 分镜图参考

## 04 视频参考

### 4.1 动作参考

模板：

```
参考「视频n」的「动作描述」，生成「画面描述」，保持动作细节一致。
```

### 4.2 运镜参考

模板：

```
参考「视频n」的「运镜描述」，生成「画面描述」，保持运镜一致。
```

### 4.3 特效参考

模板：

```
参考「视频n」的「特效描述」，生成「画面描述」，保持特效一致。
```

## 05 视频编辑

### 5.1 元素增删改

- 增加：`在「视频n」的「时间位置」+「空间位置」，增加「元素描述」。`
- 删除：`删除「视频n」中的「元素」，视频其他内容保持不变。`
- 修改：`将「视频n」中的「被替换元素」，替换为「新元素」。`

### 5.2 视频延长

- 向后延长：`生成「视频n」之后的内容+「描述」`
- 向前延长：`向前延长「视频n」+「描述」`
- 注意：模型自动截取衔接部分合成，输入视频原有片段不会重复生成

### 5.3 轨道补齐

模板：

```
「视频1」+「过渡画面描述」+接「视频2」+「过渡画面描述」+接「视频3」
```

- 最多支持3段视频输入，总时长不超过15秒
