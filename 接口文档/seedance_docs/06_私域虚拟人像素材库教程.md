# 私域虚拟人像素材资产库使用指南

> 来源: <https://www.volcengine.com/docs/82379/2333565?lang=zh>

> **警告**：仅限邀测用户。

## 概述

为确保创作者能充分利用 Seedance 2.0 强大的视频生成能力，同时规避 AI 生成内容的潜在风险，方舟推出了私域可信素材库。

## 素材资产库结构

- **Asset Group（素材资产组合）**：管理单元，可将同一虚拟人物素材放入同一组合
- **Asset（素材资产）**：单个素材文件（图像/视频/音频），可直接用于推理

### 素材状态流转

Processing -> Active（可用） / Failed（失败）

### 图像素材要求

- 格式：jpeg/png/webp/bmp/tiff/gif/heic/heif
- 宽高比(宽/高)：(0.4, 2.5)
- 宽高(px)：(300, 6000)
- 大小：<30MB

### 人像最佳实践

- 全身参考图：竖版，人物全身正面
- 人脸特写图：竖版，正面无表情特写，肩部以上，面部占画面2/3

## API 接口（需 AK/SK 鉴权）

### 创建接口

- **CreateAssetGroup**：创建素材组合
- **CreateAsset**：上传素材（异步，返回ID后用GetAsset查状态）

### 管理接口

- ListAssetGroups / ListAssets / GetAsset / GetAssetGroup
- UpdateAssetGroup / UpdateAsset
- DeleteAsset / DeleteAssetGroup

### 限流

| 接口 | 限流 |
|------|------|
| CreateAssetGroup | 10 QPS |
| CreateAsset | 300 QPM |
| ListAssetGroups / ListAssets | 10 QPS |
| GetAsset | 100 QPS |
| GetAssetGroup | 10 QPS |
| UpdateAsset / UpdateAssetGroup | 10 QPS |
| DeleteAsset | 10 QPS |
| DeleteAssetGroup | 5 QPS |

### 素材URI格式

```
asset://<asset_ID>
```

### 提示词指代方式

使用"图片1"、"视频1"等序号指代，不要直接使用 Asset ID。

### 完整使用流程

#### 1. 创建素材组合 (CreateAssetGroup)
请求参数：
- **Name**: 素材组合名称
- **Description**: 文字描述
- **GroupType**: 默认 `AIGC`（虚拟人像）
- **ProjectName**: 资源项目名称，默认 `default`

返回示例：
```json
{"Id": "group-20260318033332-*****"}
```

#### 2. 上传素材 (CreateAsset)
请求参数：
- **GroupId**: 必填，素材组合ID
- **URL**: 必填，可访问的图片/视频/音频URL
- **AssetType**: 必填，`Image`/`Video`/`Audio`
- **Name**: 选填，素材名称（仅用于ListAssets模糊搜索）
- **ProjectName**: 选填，默认`default`

返回示例：
```json
{"Id": "asset-20260318071009-*****"}
```

> **注意**：CreateAsset为异步接口，需轮询GetAsset查询状态直到Active。

#### 3. 查询素材状态 (GetAsset)
请求参数：
- **Id**: 素材ID
- **ProjectName**: 项目名称

返回示例：
```json
{
  "GroupId": "group-20260318033332-*****",
  "Status": "Active",
  "CreateTime": "2026-03-18T03:57:10Z",
  "AssetType": "Image",
  "UpdateTime": "2026-03-18T03:57:14Z",
  "ProjectName": "default",
  "Id": "asset-20260318035710-*****",
  "Name": "",
  "URL": "https://ark-media-asset-stg.tos-cn-beijing.volces.com/..."
}
```

> URL 有效期为 12 小时。

#### 4. 查询素材列表 (ListAssets)
请求参数：
- **Filter.GroupIds**: 组合ID数组
- **Filter.GroupType**: `AIGC`
- **Filter.Statuses**: 状态数组 (`Active`/`Processing`/`Failed`)
- **Filter.Name**: 模糊搜索
- **PageNumber/PageSize**: 分页
- **SortBy/SortOrder**: 排序

#### 5. 查询素材组列表 (ListAssetGroups)
请求参数：
- **Filter.Name**: 模糊搜索组合名称
- **Filter.GroupIds**: 组合ID数组
- **Filter.GroupType**: `AIGC`
- **PageNumber/PageSize**: 分页

### 项目隔离说明
- 素材库按项目(Project)隔离
- Asset 和 Asset Group 的 ProjectName 必须一致
- 素材的 ProjectName 需与调用视频生成API的 API Key 所属 ProjectName 一致
- 默认 ProjectName 为 `default`

### 常见问题
**Q: 素材上传成功后无法生成视频？**
A: 可能是 ProjectName 不一致。确保 CreateAsset、GetAsset 和视频生成 API 使用同一项目。
