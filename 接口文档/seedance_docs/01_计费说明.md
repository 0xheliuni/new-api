### 计费说明

- 仅对成功生成的视频计费。因审核等原因导致生成失败的，不收取费用。
- **视频价格估算公式**：按 token 单价 × token 用量
- **正常视频 token 用量估算**：`(输入视频时长+输出视频时长) × 输出视频的宽 × 输出视频的高 × 输出视频的帧率 / 1024`
  - 注意：存在输入视频时，Seedance 2.0 和 Seedance 2.0 fast 模型针对不同的视频输出时长存在最低 Token 用量限制。
- **Draft 视频（仅480p）token 用量估算**：正常视频 token 用量公式 × 折算系数
  - Seedance 1.5 pro 的 token 折算系数：无声 0.7；有声 0.6，其他模型暂不支持。
- **准确 token 用量**：以调用 API 后返回信息中的 usage 字段为准。
